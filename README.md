# IONIC UI SERIES

## Drone Store Mobile Application

# Made with :
 * Ionic
 * Angular Typescript
 * HTML5
 * SCSS

# How to run

 * clone the repo
 ````bash
 
 git clone https://github.com/stevehoober254/saleDrone 
 
 ````
 * Run npm install
 
 ````bash
 npm install
 
 ````
 * Run npm serve  
 
 ````bash
 
 ionic serve
 
 ````
